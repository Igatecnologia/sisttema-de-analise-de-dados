📚 Como o custo de IA é calculado

Toda chamada à OpenAI cobra por token — não por requisição, não por minuto, não por mês. Cada palavra em português ≈ 1.3 tokens. Input (o que você envia) custa 4× menos que output (o que a IA responde).

PASSO 1 · INPUT

O que é enviado para a IA

  • 📝 System prompt ~1.500 tokens (instruções, schema das tools)
  • 💬 Histórico ~500 tokens (últimas 5 mensagens)
  • ❓ Pergunta atual ~50 tokens
  • 📊 Resultado das tools ~1.000 tokens (dados do Postgres)
≈ 3.000 tokens input = $0.00045
PASSO 2 · OUTPUT

O que a IA gera de resposta

  • 🔧 Tool calls ~200 tokens (decisão de quais ferramentas chamar)
  • 💭 Reasoning ~300 tokens (raciocínio interno)
  • 📨 Resposta final ~1.000 tokens (texto + sugestões)
≈ 1.500 tokens output = $0.00090
PASSO 3 · TOTAL

Custo de UMA pergunta

$0.00045 (input)
+ $0.00090 (output)
────────────────
= $0.00135 por chamada
≈ R$ 0,007 por pergunta · ou ~150 perguntas por R$ 1

⚡ Estratégias para reduzir custo (até 60%)

🗄️

Prompt caching automático

OpenAI cacheia automaticamente prompts > 1024 tokens repetidos. O system prompt + schema das tools é o mesmo sempre — entra em cache na 2ª chamada.

Input cached: $0.075 (50% off)
Economia esperada: ~30-40%
💾

Cache de respostas em Redis

Perguntas idênticas em janela de 5 minutos retornam direto do cache. Útil para "quanto vendi hoje?" repetida várias vezes ao dia.

Hit rate esperado: 15-25%
Economia: ~20%
🎯

Routing por complexidade

Classificador rápido escolhe entre Gemini Flash ($0.075) para queries simples e gpt-4o-mini para complexas.

Mix 70% Gemini / 30% OpenAI
Economia: ~50%
📦

Batch API (50% off)

Relatórios agendados não-urgentes via Batch API da OpenAI: processamento em até 24h por metade do preço.

Aplicável: ~30% do volume
Economia: ~15%
🔧

Tool result truncation

Resultados grandes do Postgres limitados a top-50 linhas + total. Reduz input em 60-80% sem perder qualidade da resposta.

Aplicado em 100% das tools
Economia: ~25%
🎁

Free tier Groq como fallback

Cliente Beta com perfil Light pode usar Groq (free tier 14k req/dia) como provider primário até virar Médio.

~60% dos clientes Light
Economia: $0 nesses

🔍 RAG e Embeddings (custo separado)

O que indexamos no pgvector

O Copilot V2 (Python) indexa documentos do tenant em ai_documents: fichas técnicas, descrições de produtos, notas de cliente, FAQs internos. Cada documento vira embedding de 1024 dimensões via text-embedding-3-small.

Item Quantidade típica Tokens Custo único
Fichas técnicas indexadas~500200k$0.004
Catálogo de produtos~2.000600k$0.012
Histórico de notas/audit~10.0002M$0.040
Total indexação inicial~12.500~3M$0.06 / cliente

Reindexação incremental ~$0.005/mês por cliente. Storage pgvector: 1024 dim × 4 bytes × 12.500 = 50 MB · negligível no Supabase Pro 8 GB.

Por que vale a pena

  • 📈 +40% qualidade em queries sobre produtos específicos
  • 🎯 Reduz alucinações — responde só com fato indexado
  • 🔒 Contexto privado — RLS garante isolamento
  • ⚡ Busca híbrida — semântica + tsvector full-text
Custo total RAG por cliente:
• Inicial: ~$0.06 (uma vez)
• Mensal: ~$0.005
• Negligível: < R$ 0,03/mês

🛡️ Governance, telemetria e privacidade

📊

Telemetria por tenant

Tabela ai_usage registra cada chamada: tokens in/out, custo USD, latência, modelo, hash da pergunta, erro. Visível em /super-admin/ai-usage com agregação mensal.

🛡️

Budget alerts + hard cap

Tenant define limite mensal (default R$ 50). Webhook + email quando atinge 80%. Em 100% bloqueia chamadas até o admin desbloquear. Evita susto no faturamento.

🔒

Privacidade e LGPD

OpenAI não treina com nossos dados (default API). Dados sensíveis (CPF, senhas) filtrados antes do prompt via regex. RLS garante que tenant A nunca veja embedding de tenant B no RAG.

💡 Por que dá pra escalar barato
  • • Custo IA é linear por cliente (R$ 3,45/mês) — sem surpresas
  • • Infra fixa (Render, Supabase, Sentry) dilui com volume
  • • Margem bruta Pro: 81% · acima do benchmark SaaS B2B (75%)
  • • LTV/CAC = 5,3× (saudável é > 3×)
🚨 Riscos a vigiar
  • • Power users de IA (R$ 150/mês) precisam estar em Enterprise
  • • Stripe live precisa CNPJ + KYC antes de cobrar
  • • Pentest é bloqueador GA (R$ 10-30k) — agendar já
  • • Câmbio USD: cada R$ 0,50 de alta = +R$ 50-100/mês na conta
Conclusão: infraestrutura é barata e escala bem (R$ 660 → R$ 843 indo de 5 → 50 clientes). O grande gasto é one-time pré-GA (~R$ 25k), pago no realista pelos primeiros 4-6 meses de receita. Em 12 meses, esperança ponderada: R$ 103k em margem acumulada.
